# Y1hy11 Tech Blog

A modern tech blog built with React, focused on delivering insightful IT news and industry analysis.

📖 [Visit the Blog](https://y1hy11.github.io/blog/)

## Overview

A personal blog platform sharing insights and analysis about:

- 🔒 Cybersecurity
- ☁️ Cloud Computing
- 🤖 AI & Machine Learning
- 🌐 Web Development
- 📱 Mobile Development
- 🔧 DevOps
- 💡 Tech Industry Trends

## Features

- 🎯 In-depth technical articles
- 📱 Responsive design for all devices
- 🔍 Category-based content organization
- 💬 Interactive comment system
- ⚡ Fast loading and smooth navigation
- 🌙 Dark mode support

## Blog Categories

📚 Main categories you'll find on the blog:

- 📘 Tutorials & Guides
- 🎓 Technical Deep Dives
- 📊 Industry Analysis
- 💭 Tech Opinion Pieces
- 🔮 Future Tech Trends
- 💻 Code Snippets & Examples

## Tech Stack

Built with modern web technologies:

- React 19
- React Router DOM 7
- Framer Motion
